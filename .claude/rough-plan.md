# Task Create a Freshbooks timepicker Raycast extension

The goal of this extension is essentially replicate the same Freshobooks timepicker as a raycast extension.

## Details

The extension should let you easily start/stop a timer and when stopped it should let you submit what it was for attaching it to a project and service and passing it notes of what was done. It should ensure that we have the correct time and date so it logs correctly in freshbooks via api.
